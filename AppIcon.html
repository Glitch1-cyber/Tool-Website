How to Convert a Python file to an App

Home Page

Step 1:

Installation

  • Install Python from python.org
  • After installing python, Install the ZIP File from the Home page
  • It should look like this:

Step 2:

Creation

  • Unzip(Extract) the Installed ZIP File to an empty folder in your Documents.
  • You should see this in the folder:
  • Now you need to create a Shortcut of the File 'TOOL.py'. Right-click the file and select 'Show more options', then click 'Create Shortcut'.
  • After creating the shortcut, you should see two 'TOOL.py' files.

Step 3:

Designing App

  • Cut the shortcut of the file TOOL.py and paste it on your Desktop.
  • Right-click the file on your Desktop and click 'Properties'.
  • Click on 'Change Icon' and then click 'Browse'.
  • Select the 'anonymity.ico' icon from the folder you extracted the files to and click OK.
  • Finally, rename the app by right-clicking it and choosing 'Rename'. Give it the name you want.
